# Getting Started

**Product:** SalesEdge — Intelligent Sales & Revenue Operations Platform  

[← Documentation index](../README.md) · [Executive cockpit](executive-cockpit.md)

---

## First login

1. Navigate to the SalesEdge URL provided by your administrator.  
2. Sign in with your **corporate identity** (mechanism depends on deployment: JWT demo vs SSO).  
3. On success, you land on the **dashboard** with data scoped to your role.

## Dashboard overview

- **Pipeline health** — deal risk and stage distribution.  
- **Prospect quality** — fit scores for recent leads.  
- **Retention** — churn risk highlights when enabled.  
- **Macro strip** — India rates/FX context where connectors are active.

## Navigation guide

- Use the primary nav for **Dashboard**, **Prospects**, **Deals**, and **Settings** (labels may vary by build).  
- **Command palette** (if enabled) jumps to entities quickly.  
- **Notifications** surface agent-generated signals.

## Key concepts

| Concept | Meaning |
|---------|---------|
| **Fit score** | How well an account matches ICP (0–100) |
| **Risk score** | Deal danger (higher = more at risk) |
| **Churn probability** | 90-day estimated churn likelihood |
| **Recovery priority** | Ranked list combining risk, value, recoverability |
| **Signal** | Event routed to an AI agent (e.g. engagement drop) |

All currency values display in **INR**; timestamps default to **IST** in the UI.
